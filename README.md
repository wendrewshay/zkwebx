# zkwebx
基于淘宝zkWeb用springboot重构的项目，可以方便管理zookeeper  


1.获取代码到IDE后，修改zk.properties中的zookeeper服务地址，双击启动类调试；  
2.部署打包：mvn -Dmaven.test.skip clean pacakage，需要打成war包部署；  
3.访问：http://ip:port/zkwebx-1.0
